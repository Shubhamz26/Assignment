Interpreter:

It is a computer program used for converting high-level program statements into machine codes.
Interpreter translates only one statement at a time of program.
They create an exe of the programming language before the program runs.


Compiler:

It is a computer program used to transform codes written in a programming language into machine code.
It check all types of errors,ranges and limits.
The run time of its program is longer, and it occupies more memory.
